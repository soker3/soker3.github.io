如何更新到github上

Step1: 在github上新建仓库

  • 新建仓库create a new repository
  • 仓库名称一定为：自己昵称.github.io

Step2: 把本地项目部署到github上

  • 设置github地址，也可直接更改config.toml文件，把baseUrl设置为github的仓库地址：
hugo --baseUrl="https://soker3.github.io/" --buildDrafts
  • 其他配置项：
baseURL = "https://example.org/"
languageCode = "en-us"
title = "My New Hugo Site"
theme = "ananke"

Step3: 把public传到github上

git init
git add .
git commit -m "init"
git remote add origin https://github.com/soker3/soker3.github.io.git
git push -u origin master
